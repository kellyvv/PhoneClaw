iPhone ローカル AI Agent

PhoneClaw for iPhone

PhoneClaw は iPhone 向けのモバイルネイティブなローカル AI Agent と Agent フレームワークです。オンデバイスモデル、ネイティブ iOS Skills、権限境界、Live / LiveLand、任意の LAN ベース Mac 推論を組み合わせます。

TestFlight に参加 GitHub を見る

これは iPhone 向け PhoneClaw であり、同名の Android アプリとは関係ありません。

Positioning

クラウドチャットのラッパーではなく、電話から設計された Agent です。

PhoneClaw は iPhone を単なる画面ではなくランタイムとして扱います。モデル読み込み、Skill ルーティング、権限確認、メモリ予算、ダウンロード、Live 入口、タスク状態はモバイル制約を前提に設計されています。

オンデバイスモデル

Gemma 4 E2B / E4B を LiteRT-LM 経由で iPhone 上で実行し、MiniCPM-V 4.6 を画像理解と LIVE カメラ向けに使います。

ネイティブ iOS Skills

Calendar、Reminders、Contacts、Clipboard、HealthKit、翻訳、画像、明示的な Web Search を権限境界内で扱います。

Mac Gateway

より重い推論が必要なときは、同じ LAN 上の Mac を任意の推論ノードとして使えます。iPhone は Agent 体験と権限境界を維持します。

Privacy

デフォルトではアップロードしません。

チャット、画像、個人データはデフォルトで iPhone 上に残ります。Web Search、URL の読み取り、またはペアリングした Mac へのリモート推論を明示的に使う場合だけ、関連データが iPhone の外へ出ます。
