Local by default

Live interaction still follows the same privacy model.

LiveLand and LIVE mode run on local models by default. The local path is fully offline; Web Search, URL reading, and optional Mac remote inference are explicit user-triggered network paths.

Local task status

The phone shows what the agent is doing as local task state.

Safer actions

Model-aware
