PhoneClaw for iPhone

PhoneClaw 是面向 iPhone 的移动端本地 AI Agent 和 Agent 框架：端侧模型、iOS 原生 Skills、权限边界、Live / LiveLand 交互，以及可选的局域网 Mac 远程推理。

这是 PhoneClaw for iPhone，一个 iOS 端侧 Agent 项目，与同名 Android 应用无关。

定位

面向 iPhone 运行时设计的移动端 Agent 框架。

PhoneClaw 把 iPhone 当成 Agent 运行时。模型加载、Skill 路由、权限确认、内存预算、模型下载、Live 入口和任务状态都围绕移动端约束设计。

端侧模型

Gemma 4 E2B / E4B 通过 LiteRT-LM 在 iPhone 本地运行，MiniCPM-V 4.6 用于图片和 LIVE 相机场景。

原生 iOS Skills

Calendar、Reminders、Contacts、Clipboard、HealthKit、翻译、图片和显式 Web Search 都在权限边界内执行。

Mac Gateway

需要更重推理时，可以把同一局域网内的 Mac 作为可选推理节点，iPhone 仍保留 Agent 体验和权限边界。

隐私

数据留在设备端，联网能力显式触发。

聊天、图片和个人数据默认留在 iPhone。联网搜索、打开网页和配对 Mac 远程推理是用户显式触发的能力。PhoneClaw 的本地路径支持完全离线运行。
